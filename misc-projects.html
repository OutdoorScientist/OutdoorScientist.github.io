Custom Human Interactive Device (HID) Embedded System

I created a custom ergonomic computer mouse to alleviate wrist pain.

I programmed a Raspberry Pi Zero W to function as a human interface device (HID) when connected to a computer. By pairing an Xbox controller with the Raspberry Pi via Bluetooth, I captured its inputs and converted them into binary signals for mouse and keyboard actions. This allowed the Xbox controller to operate as both a mouse and keyboard, with the Raspberry Pi managing the conversionâ€”no additional software required on the host computer. I mapped the controller buttons to keyboard shortcuts and the joysticks to mouse movements.

This unique setup, with button-to-shortcut mapping and the lack of software dependency on the host computer, distinguishes my project from other similar devices. The entire project was developed using Python and shell scripts.

During this project, I accomplished the following:

  • Developed a unique ergonomic mouse and keyboard system that functions universally without requiring additional software on the host computer, utilizing Python, shell scripting, binary Bluetooth descriptors, a Raspberry Pi Zero W, and an Xbox controller.
  • Gained an understanding of how human interface devices (HIDs) communicate with computers.
  • Learned how to interpret and use binary inputs for both keyboards and mice.
  • Set up a Raspberry Pi to register as a HID on power up through automatic execution of scripts.
  • Developed a custom Python library to remap all the outputs of an xbox controller into mouse movements and keyboard inputs
  • Learned the distinction between power cables and data cables and their appropriate applications.

Skills: Human Interactive Devices, Bluetooth, Embedded Systems, Raspberry Pi, Python

Medical Receipt Manager Embedded System

Automated the scanning and tracking of health-related receipts for efficient management and future integration with a machine learning model.

I recently opened a Health Savings Account (HSA), which requires me to track my health-related purchases for future reimbursement. To simplify this process, I decided to automate the tracking of my receipts. I connected a printer/scanner to a Raspberry Pi 3B, allowing me to control the device using terminal commands. Now, when the Raspberry Pi boots up, a Bash script monitors the GPIO pins, and pressing an external button triggers the scanner. The scanned file is automatically saved to a hard drive with a unique, ascending filename. This setup makes it easy to store, manage, and track receipts for medical purchases. Additionally, I plan to incorporate a machine learning model for image-to-text extraction, which will populate a CSV file with relevant receipt data, further streamlining the tracking process.

  • Gained a deeper understanding of Linux operating systems and how to monitor GPIO pins.
  • Successfully created dynamically named and organized directories on external storage devices through Bash scripts triggered on external physical button presses.
  • Gained knowledge on various printer drivers and open-source software to enable their seamless integration with Linux operating systems.

Skills: Linux, Embedded Systems, GPIO, Raspberry Pi
