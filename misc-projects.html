Custom Computer Mouse

My wrist was hurting so I decided to build a personal, ergonomic computer mouse. I successfully achieved by programming a raspberry pi zero W to be register as a human interactive device (HID) when plugged into a computer. I was then able to connect an xbox controller to the raspberry pi zero through bluetooth. Then I was able to capture the inputs from the xbox controller and convert them into binary mouse and keyboard inputs. The xbox mouse outputs were translated into mouse and keyboard computer inputs by the raspberry pi. No additional software was needed on the host computer making it a universal device.

  • Learned how human interactive devices communicate with a computer.
  • Learned the binary inputs for a keyboard and mouse.
  • Learned how to setup a raspberry pi to execute scripts on boot.
  • Learned how to setup a custom python library for shared functions.
  • Learned the difference between power cables and data cables.

Medical Receipt Manager

I recently opened a Health Savings Account (HSA) which requires me to keep track of my health related purchases in order to be reimbursed at a later date. I decided to investigate ways I could automate the tracking of these receipts in order to better manage everything. I ended up connecting a printer/scanner to a Raspberry Pi 3B in which I could control the printer through terminal commands. Upon boot of the PI a python script is executed that monitors a button input. With the push of this button, the scanner is triggered, and the resultant scan is saved with an ascending filename to a hard drive.

  • Learned more about the Raspbian OS, creating directories through terminal commands, verifying directories exists, and accessing external storage from the terminal.
  • Learned about the drivers of printers and the open source software to connect through them on Linux.
