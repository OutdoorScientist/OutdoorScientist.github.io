Backend Projects

I have worked extensively with both AVEVA Insight and AVEVA CONNECT, contributing to the Azure backends to enable real-time industrial data visualization and monitoring. My work involved backend optimizations, schema modifications, data identifier implementation, and performance improvements.

Additionally, I oversaw the AWS deployment of a machine learning-based SaaS platform for Texas school leaders, managing key AWS services such as EC2, RDS, S3, and IAM roles while ensuring legal compliance for student data.

I also have experience setting up databases for various personal projects, including trivia databases and a movie recommendation platform.

AVEVA's Azure Backend

Worked on both AVEVA Insight and AVEVA CONNECT Azure backends in order to enable industrial operations to visualize, monitor, compare, and modify varying data structures in real time.

  • Learned how the AVEVA Azure backend was setup, accessed, secured, and how to write query data from it within Angular.
  • My backend contributions included updating the site logo and name, altering data schemas, adding unique data identifiers, modifying build pipeline steps, adding eslint rules, verifying data calls, and investigating ways to optimize backend data calls to improve performance.

ELRC's machine learning SaaS

My role was to enable Texas school leaders to gauge their school's STAAR performance through the use of a novel machine learning model, with over 90% accuracy, in order to optimize their school's test preparation. I achieved this by overseeing the AWS back-end deployment of the ELRC's software as a service (SaaS) website.

  • This deployment included the investigation, tier selection, and implementation of Amazon Web Services (AWS) EC2, Relational Database Service (RDS), S3 Bucket, and Lambda services.
  • I personally managed the Identity and Access Management (IAM) roles, EC2s configuration, S3 Bucket setup and connection, and the selection of the type of database used ensuring legal compliance for all services due to student data.
