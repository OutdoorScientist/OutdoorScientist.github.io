Heavy Duty Robotic Arm Vision

I researched, designed, and started construction of a heavy duty robotic arm that loads my dishwasher with dirty dishes pulled from the sink. I was able to achieve this through the combination of a custom object identification machine learning model and stereo imaging for depth data.

  • Captured, organized, and annotated hundreds of images of dirty dishes in order to train a machine learning object identification model
  • Figured out a way to use the previous models to annotate the images in which I could review them in order to speed up data generation and accuracy.
  • Learned how to train models off of already trained models in order to enable a single machine learning model to identify multiple items.
  • Went through a series of revisions as I discovered limitations and model needs in order to achieve the defined objective. For example, I discovered the need to identify the sink and counter due to the need for environment awareness for the robotic arm.
  • Successfully trained and validated an 89% accurate model that could identify the counter, sink, mugs, cups, bowls, silverware, and plates of varying sizes and colors.

ELRC's Machine Learning SaaS

My role was to enable Texas school leaders to gauge their school's STAAR performance through the use of a novel machine learning model, with over 90% accuracy, in order to optimize their school's test preparation.

  • Successfully deployed a SaaS with an integrated ML model to AWS.
  • Learned to clarify with clients how to organize student training data in order to accelerate model training.
  • Implemented SciKit-Learn machine learning model with margin of error.

Webcam Assistance

I created an accessible Webcam Adjustment Site that helps those with visual impairments (i.e. astigmatism, blind, etc..) setup their webcams by themselves.

  • Implemented robust logic to make accessible recommendations based on confirmed webcam input, room lighting, the webcam's vertical and horizontal tilt, and the centering of the feed on the main person.
  • These recommendations are derived using a facial recognition model, image-based python libraries, and intermediate math.
