Heavy Duty Robotic Arm

I researched, designed, and started construction of a heavy duty robotic arm that loads my dishwasher with dirty dishes pulled from the sink. I was able to acheive this thorugh the combination of a custom object idenitification machine learning model and stero imaging for depth data.

  • Captured, oragnized, and annotated hundreds of images of dirty dishes in order to train a machine learning obect idetnification model
  • Figured out a way to use the previous models to annotate the images in which I could review them in order to speed up data generation and accuracy.
  • Learned how to train models off of already trained models in order to enable a single machine learning model to idenitfy multiple items.
  • Went through a series of revisions as I deiscovered limitations and model needs in order to acheive the defined objective. For example, I discovered the need to idenifty the sink and counter due to the need for environment awareness for the robotic arm.
  • Successfuly trained and validated an 89% accurate model that could identify the counter, sink, mugs, cups, bowls, silverware, and plates of varying sizes and colors.

Webcam Assistance

I created an accessible Webcam Adjustment Site that helps those with visual impairments (i.e. astigmatism, blind, etc..) setup their webcams by themselves.

  • Implemented robust logic to make accesible recommendations based on confimred webcam input, room lighting, the webcam's vertical and horizontal tilt, and the centering of the feed on the main person.
  • These recommendations are derived using a facial recognition model, image-based python librarires, and intermediate math.
